# Frontend Design Skill

This skill provides guidelines for creating distinctive, production-grade frontend interfaces with exceptional design quality.

## Purpose

Guides the creation of memorable, polished web interfaces that avoid generic AI aesthetics. Focuses on:
- Bold aesthetic choices
- Production-ready code
- Creative typography and color
- Thoughtful animations and interactions
- Distinctive spatial composition

## Usage

The skill is automatically applied when building frontend components, pages, or applications. It emphasizes:
1. Understanding context and purpose
2. Committing to a clear aesthetic direction
3. Implementing with precision and attention to detail
4. Avoiding generic design patterns

## Key Principles

- Choose distinctive, characterful fonts
- Commit to cohesive color schemes with CSS variables
- Use animations purposefully for high-impact moments
- Create unexpected layouts with asymmetry and creative composition
- Add atmospheric backgrounds and visual details
- Match implementation complexity to aesthetic vision
